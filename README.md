# Creacion-de-modulo-operacion.py-
módulo en Python llamado Operaciones.py que contenga las funciones básicas de una calculadora. Dentro del módulo deberás definir las siguientes funciones:

sumar(a, b)
restar(a, b)
multiplicar(a, b)
dividir(a, b): En esta función, asegúrate de manejar el caso en que el divisor sea cero.
Una vez creado el módulo, deberás programar un archivo principal llamado main.py que:

Importe el módulo Operaciones.
Solicite al usuario dos números.
Solicite la operación que desea realizar (suma, resta, multiplicación o división).
Utilice la función correspondiente del módulo Operaciones para obtener el resultado.
Muestre el resultado por pantalla.
